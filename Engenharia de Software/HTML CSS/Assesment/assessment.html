Multimídia na Web

1 -O elemento source permite elementos como src e type, ele suporta arquivos como mp3, ogg e wav. Como outros atributos podemos citar o autoplay, que realiza a execução do arvquivo assim que a pagina é carregada, o controls, que adiciona controles ao player de audio, o mute que muta o player, o loop que faz o player recomeçar do inicio assim que chegar ao fim.

2 -O elemento source permite elementos como src e type, ele suporta arquivos como mp4, ogg e webm. O elemento video recebe atributos como altura e largura para definirmos o tamanho do player.

3 -Uma linguagem de estilização, tal como css, ou o uso de apis, como canva. Apis são aplicaões que possibilitam a criação de funcionalidades complexas de maneira mais facil.

4 -Pode ser aplicado usando atributos como altura e largura para definir o tamanho de um player por exeplo, elemento hover para que o mesmo receba destaque, ou até mesmo elementos como o position: fixed;.
